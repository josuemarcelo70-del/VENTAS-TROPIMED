🌴 Configurar Sincronización

Para usar la app en múltiples dispositivos en tiempo real, necesitas conectarla a tu proyecto de Firebase (gratuito).

Pasos rápidos:
1. Ve a console.firebase.google.com
2. Crea un proyecto → Agrega una app web
3. Copia el valor de projectId de tu config
4. En Firestore Database → Reglas → pega:
allow read, write: if true;
